1. Go to: https://app.netlify.com
2. Log in with GitHub or Email
3. **Drag the entire `website` folder** onto the Netlify dashboard
4. That's it! 🎉 Your site is live!

1. Go to: https://vercel.com/drop
2. Drag the `website` folder onto the page
3. Done!
